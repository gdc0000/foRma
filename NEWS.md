# foRma (version 1.0.0)

* Initial CRAN release
* AI-powered SEM visualization using Google Gemini API
* Static ggplot2-based SEM plots with APA formatting
* Interactive visNetwork-based SEM plots

* Support for both latent and observed variables
* Significance-based path styling (solid/dashed lines)
